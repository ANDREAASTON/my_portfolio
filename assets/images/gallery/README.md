# Gallery Thumbnail Images

## Overview
Project category thumbnail images displayed in the main portfolio gallery on the homepage.

## Files
- `Graphics.jpg` - Graphic Design project thumbnail
- `smartBin.jpg` - Smart Bin project thumbnail
- `Photography.jpg` - Photography project thumbnail
- `RescueRobot.jpg` - Rescue Robot project thumbnail
- `Robotics Instructor.jpg` - STEM Workshops project thumbnail

## Usage
- Project cards on homepage (index.html)
- Projects section gallery
- Navigation to detailed project pages

## Dimensions & Specifications
- Display size: 350px width (responsive)
- Aspect ratio: 16:9 or square
- Format: JPG (optimized)
- File size: <150KB each

## To Update or Add Thumbnails
1. Choose representative image for project
2. Create thumbnail (350-400px width)
3. Optimize for web (compress to <150KB)
4. Name clearly: `[project-name].jpg`
5. Update src in project-card
6. Test on all devices

## Adding New Project
1. Create thumbnail image (350x200px minimum)
2. Save as `new-project-name.jpg`
3. Create project card HTML in index.html
4. Link to project details page
5. Test responsive design
6. Update this README

## Tips
- Use high-contrast, visually appealing images
- Keep consistency with existing thumbnails
- Test visibility on mobile devices
- Consider hover effects in CSS
- Optimize for fast loading
