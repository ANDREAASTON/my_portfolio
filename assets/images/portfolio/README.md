# Portfolio Images

## Overview
Profile and personal branding images used in the portfolio.

## Files
- `AndreaAston.jpg` - Main profile photo used in About section

## Usage
- Profile picture in About section (index.html)
- Main portfolio branding
- LinkedIn and social media profile images

## Notes
- Recommended dimensions: 400x500px
- Format: JPG
- Optimized for web display

## To Add New Images
1. Save high-quality original
2. Optimize for web (reduce file size)
3. Use descriptive naming: `profile-[version].jpg`
4. Update HTML references
5. Test on mobile devices
