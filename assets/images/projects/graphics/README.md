## To Add New Designs
1. Export design as JPG (high quality)
2. Optimize for web (compress to <500KB)
3. Use naming convention: `image26.jpg`, `image27.jpg`, etc.
4. Or rename with descriptive names: `social-media-design-2025.jpg`
5. Add to gallery in graphicDesigning.html
6. Update this README

## Best Practices
- Keep consistent aspect ratios
- Ensure designs are representative samples
- Include variety (posters, social media, branding, etc.)
- Test gallery responsiveness after adding

## Recommended Workflow
1. Design in Adobe Photoshop/Illustrator
2. Export as JPG at 72DPI
3. Compress with TinyPNG
4. Save to this folder
5. Update gallery HTML
6. Test on multiple devices
