# Project Videos

## Overview
Location for all project video demonstrations and documentation.

## Subdirectories
- `projects/smartbin/` - Smart Bin project videos

## To Add Videos for New Projects
1. Create subfolder: `projects/[project-name]/`
2. Add README.md with video specifications
3. Place video files in the folder
4. Update HTML to reference new path
5. Test playback across browsers

## Video Specifications
- Format: MP4 (H.264/AAC codec)
- Max file size: 20MB
- Resolution: 720p (1280x720) minimum
- Frame rate: 30fps
- Bitrate: 1500-2500 kbps

## Recommended Projects for Video
- Smart Bin: Assembly/demo process
- Rescue Robot: Navigation and obstacle avoidance
- Graphics: Design timelapse
- Photography: Portfolio showcase
- Arduino Workshops: Student project highlights

## Workflow
1. Record video
2. Edit (trim, add captions if needed)
3. Export as MP4
4. Compress for web
5. Save to appropriate project folder
6. Update HTML references
7. Test on multiple devices
8. Document in README.md

## Tools
- Recording: OBS, ScreenFlow, Camtasia
- Editing: Adobe Premiere, Final Cut Pro, DaVinci Resolve
- Compression: ffmpeg, HandBrake, CloudConvert
- Hosting: Self-hosted, Cloudinary, or Bunny CDN (optional)
